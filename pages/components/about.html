This web application runs entirely in your browser using WebAssembly (https://webassembly.org). All data is processed locally in your browser and is not transmitted anywhere. This benefit comes at the cost that the initial loading might take a while since we need to download all the resources (that would otherwise live on a server). If the application behaves unexpectedly, try clearing your browser cache and reloading the page (CTRL + F5 in most browsers). Clearing the cache helps because an update might have changed files that are now in conflict with the files in your browser cache.
